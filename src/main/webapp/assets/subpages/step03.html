Integrating with HTML

Now that we have a simple JavaScript application, we would like to use it in an HTML page. To do this, we need two steps:

1.- Generate JavaScript

With the sbt fastOptJS task, we can generate a single JavaScript. This will perform some fast optimizations and generate the target/scala-2.11/lovescalajs-fastopt.js file containing the JavaScript code.

2.- Create the HTML Page

To load and launch the created JavaScript, you will need an HTML file. Create the file demo-page.html in the project root with the following content:

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>Love Scala.js Demo</title>
</head>
<body>
<script type="text/javascript" src="./target/scala-2.11/lovescalajs-fastopt.js"></script>
<script type="text/javascript">
    demo.webapp.LoveScalaJsApp().main();
</script>
</body>
</html>

If we open the page, we can see in the JavaScript console: 

Hello world!
