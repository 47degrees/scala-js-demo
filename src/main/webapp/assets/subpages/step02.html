Easily integrated into your app

A Hello World example: Firstly, we add a very simple LoveScalaJsApp app in the demo.webapp package.
Create the file src/main/scala/demo/webapp/LoveScalaJsApp.scala: 

package demo.webapp

import scala.scalajs.js.JSApp

object LoveScalaJsApp extends JSApp {
  def main(): Unit = {
    println("Hello world!")
  }
}

To run this, simply launch sbt and invoke the run task:

$ sbt
> run
[info] Compiling 1 Scala source to (...)/scala-2.11/classes...
[info] Running tutorial.webapp.TutorialApp
Hello world!
[success] Total time: 28 s, completed May 18, 2015 9:22:46 PM
Note:

So your code has actually been executed by the Rhino JavaScript interpreter which is cool because it runs out of the box, without having to install anything. But it is terribly slow. In general, you do not want to use it for your day-to-day development.

Node.js is a much more performant JavaScript engine. To run your code with Node.js, you need to install it, and set FastOptStage in your build.sbt:

scalaJSStage in Global := FastOptStage
